# Queue System

Queue scripts for sequential execution.

- **Add to Queue**: Right-click a script > "Queue".
- **Run All**: In the "Queue" panel, click "Run All" (▶▶).
- **Manage Queue**: Use "Move Up" (⬆️), "Move Down" (⬇️), or "Remove" (✖️).
- **Save/Load**: Save queues as `.txt` files or load existing ones.
